# dekosodas-backend

To install dependencies:

```bash
bun install
```

To run:

```bash
bun start
```

Unit tests:

```bash
bun test
```

# Unit test env variables

.env.test:

JWT_PRIVATE_KEY_PATH  
JWT_PUBLIC_KEY_PATH  
SESSION_SECRET

MONGO_URL  
MONGO_DB_NAME //preferably use a test db  
MONGO_CA  
MONGO_CERT
